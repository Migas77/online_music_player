# online_music_player
TP1 TPW

### Setup
```bash
# clone repo 
git clone git@github.com:Migas77/online_music_player.git
# go inside project 
cd MusicPlayer/
# create venv
python3 -m venv venv
# go inside the venv
source venv/bin/activate
# install requirements
pip install -r requirements.txt
# run
python3 manage.py runserver
```
